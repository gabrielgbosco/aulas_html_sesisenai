Explicando as Tags no HTML

Feito por: Gabriel G. Bosco

Nesta página você irá encontrar uma explicação de para que servem algumas Tags muito utilizadas no HTML.

Tag: < html >

Serve para iniciar e terminar o código em HTML.

Tag: < head >

É a parte que não é exibida no navegador da Web quando a página é carregada. Ele contém informações como title, links para (se você deseja modelar seu conteúdo HTML com CSS),
links para favicons personalizados e outros metadados (dados sobre o HTML, como quem o escreveu, e palavras-chave importantes que descrevem o documento.)

Tag: < body >

É dentro dessa Tag que fica a parte visível do site. O que o usuário vai enxergar.

Tag: < header >

Contém alguns elementos de cabeçalho mas também outros elementos como um logo.

Tag: < section >

Define uma seção em um documento HTML. Pode ser usada para agrupar conteúdo relacionado dentro de uma página.

Tag: < div >

É uma divisão genérica que é usada para agrupar elementos em um documento HTML e aplicar estilos a eles com CSS.

Tag: < footer >

Define o rodapé de uma seção ou de um documento HTML. Geralmente contém informações de copyright, links de contato, etc.

Tag: < h1 > até < h7 >

São tags de cabeçalho que definem o nível de importância e hierarquia de um título em uma página HTML, sendo < h1 > o mais importante e < h7 > o menos importante.

Tag: < hr >

É usado para criar uma linha reta horizontal para dividir parágrafos ou seções.

Tag: < p >

Define um parágrafo de texto em um documento HTML.

Tag: < ul >

Define uma lista não ordenada em HTML. Ela geralmente contém uma série de elementos < li >.

Tag: < li >

É usada para representar um item de uma lista, seja ordenada < ol > ou não ordenada < ul >.

Tag: < br >

É usada para inserir uma quebra de linha dentro de um texto ou em um contexto onde o HTML não suporta espaçamento automático.

Tag: < strong >

É usada para deixar o texto dentro dela em negrito.

Tag: < a >

É usada para criar links em HTML. O atributo "href" especifica o URL para o qual o link aponta.

Tag: < img >

É usada para inserir imagens em uma página HTML. O atributo "src" especifica o URL da imagem e o "alt" fornece um texto alternativo para acessibilidade.

Tag: < input >

É usada para criar controles interativos em formulários HTML, como campos de texto, botões, caixas de seleção, etc.

Tag: < button >

É usada para criar um botão clicável em uma página HTML.

Tag: < iframe >

É usada para incorporar conteúdo de outra página da web dentro de uma página HTML. Isso é comumente usado para inserir vídeos do YouTube, mapas do Google, etc.

Tag: < table >

É usada para criar uma tabela em HTML. Ela contém uma ou mais linhas < tr >, que por sua vez contêm células de dados < td > ou células de cabeçalho < th >.

Tag: < meta >

É usada para fornecer metadados sobre o documento HTML, como autor, descrição, palavras-chave e configurações de exibição (como a viewport para dispositivos móveis).

Tag: < aside >

É usada para conteúdo relacionado ao conteúdo principal da página, mas que pode ser considerado secundário ou complementar. Por exemplo, barras laterais com links relacionados ou anúncios.

Tag: < span >

É semelhante à < div >, mas é usada para aplicar estilos a partes menores de um texto ou para agrupar elementos em linha.
